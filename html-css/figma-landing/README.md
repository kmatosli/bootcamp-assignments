# Figma Landing Page Project

This project recreates a modern portfolio landing page from a Figma design using HTML and CSS.

## 🔍 Overview

The goal of this project was to translate a static Figma design into a fully structured and styled webpage. The layout focuses on clean visual hierarchy, consistent spacing, and modular sections.

## ✨ Features

- Hero section with navigation and profile image
- Client logo strip
- Case studies section
- Recent work section
- Testimonials section
- Contact form
- Footer

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Raleway, IBM Plex Mono)
