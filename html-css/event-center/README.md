🌍 Project Overview
The Curious Itinerary is a boutique intellectual travel studio offering 8 curated cultural immersion trips through Spain — designed for curious adults who want context, depth, and genuine human connection. Not bus marathons. Not Instagram backdrops.

🎨 Design Decisions

Navy + Cream + Gold palette: editorial, sophisticated, timeless — appropriate for an intellectual luxury travel brand
Cormorant Garamond display font: literary, refined — chosen to differentiate from generic travel site sans-serif
Zero solo upcharge messaging prominent throughout: key brand differentiator
Aire Ancient Baths comparison on pricing page: data-driven selling
Competitor comparison table with "Bus Time" column: reflects founder's lived pain point with Gate1
Pre-trip reading list featured across all trip pages: most differentiated brand asset
