# Visible API — Application Factory Pattern

Career Intelligence REST API built with Flask Application Factory Pattern.

## Setup

```bash
pip install -r requirements.txt
python flask_app.py
```

## Endpoints

- `/users` — User registration, login, profile management
- `/advisors` — Advisor registration, login, management
- `/sessions` — Career sessions, advisor assignment
- `/resources` — Career resources and tools

## Auth

JWT Bearer token required for protected routes.
